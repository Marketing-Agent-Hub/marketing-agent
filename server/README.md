AI-powered marketing agent backend with content intelligence and workflow automation.

## 🛠 Tech Stack
- **Runtime**: Node.js (TypeScript)
- **Database**: PostgreSQL (via Prisma ORM)
- **AI**: OpenAI / OpenRouter
- **Process Management**: node-cron
